## 1. 项目简介

**龙虾管家**是基于 OpenClaw 构建的 AI 生活规划助手，帮助用户智能管理日常生活。从记账、提醒到行程规划，让 AI 成为你生活中的可靠管家。

## 2. 核心功能

### 2.1 智能记账与理财
- 语音/文字快速记支出
- 月度消费分析图表
- 理财建议提醒

### 2.2 日程与任务管理
- 智能拆分大型任务
- 定时提醒（cron jobs）
- 任务进度追踪

### 2.3 健康生活提醒
- 作息时间提醒
- 喝水/运动提醒
- 健康数据记录

### 2.4 旅行与活动规划
- 行程一键生成
- 预算自动计算
- 物品清单提醒

## 3. 技术架构

```
用户消息 → OpenClaw Gateway → AI分析 → 执行动作 → 反馈结果
                                      ↓
                              Cron 定时任务
                                      ↓
                              提醒/通知推送
```

**核心技术：**
- OpenClaw Gateway（消息路由）
- Cron Jobs（定时任务）
- 浏览器自动化（可选）
